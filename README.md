# 🧠 NotesAI

An AI-powered note-taking application built using the MERN Stack and Groq AI.

## 🚀 Features

- 🔐 User Authentication (JWT)
- 📝 Create, Edit & Delete Notes
- 🔍 Live Search
- ❤️ Favorite Notes
- 🤖 AI Summarize Notes
- 📖 AI Explain Notes
- 📝 AI Quiz Generator
- 📄 Export Notes as PDF
- 📊 Dashboard Statistics
